Jac on Windows + WSL — Setup Guide

From zero → working environment: WSL (Ubuntu), Python 3.12, VS Code, Jac, byLLM (Gemini), and GitHub.

Windows 10/11 WSL (Ubuntu) Python 3.12+ Jac 0.8.x byLLM 0.4.x VS Code GitHub (SSH)

Steps

0) PowerShell (Admin) 1) Install WSL + Ubuntu 2) Update Ubuntu 3) Python 3.12 4) VS Code + Extensions 5) Project folder 6) Virtual env + pip 7) Install Jac 8) Open in VS Code 9) Hello Jac 10) byLLM + Gemini 11) Tiny API (optional) 12) Git + GitHub 13) Daily workflow 14) Troubleshooting
Windows-only (no WSL) Publish on GitHub Pages

0) Open PowerShell (Admin)

Press Win, type PowerShell, right-click → Run as administrator.

1) Install WSL + Ubuntu

Run in PowerShell (Admin):

wsl --install -d Ubuntu
Reboot if asked. On first Ubuntu launch, create a Linux username and password.

2) Update Ubuntu

Run inside the Ubuntu (WSL) terminal:

sudo apt update && sudo apt -y upgrade
sudo apt -y install build-essential git curl pkg-config

3) Install Python 3.12

python3 --version  # if < 3.12 then install:
sudo apt -y install software-properties-common
sudo add-apt-repository -y ppa:deadsnakes/ppa
sudo apt update
sudo apt -y install python3.12 python3.12-venv python3.12-dev
python3.12 --version

4) VS Code + Extensions (Windows)

winget install --id=Microsoft.VisualStudioCode -e

In VS Code → Extensions: install WSL, Python, Pylance. (Optional: GitHub Pull Requests, EditorConfig)

5) Create project (WSL)

mkdir -p ~/projects/jac-starter
cd ~/projects/jac-starter

6) Virtual env + pip (WSL)

python3.12 -m venv .venv
source .venv/bin/activate
python -m pip install -U pip

7) Install Jac (+ FastAPI stack)

pip install jaclang byllm fastapi "uvicorn[standard]"
jac --version   # expect: 0.8.x

8) Open in VS Code (WSL)

code .
VS Code → Python: Select Interpreter → choose .venv/bin/python.

9) Hello Jac

cat > main.jac <<'JAC'
with entry {
    print("Hello from Jac on WSL!");
}
JAC

jac check main.jac
jac run main.jac

10) byLLM + Gemini (typed LLM)

Get a key: Google AI Studio

echo 'export GEMINI_API_KEY="PASTE_YOUR_KEY_HERE"' >> ~/.bashrc
echo 'export JAC_MODEL="gemini/gemini-1.5-flash-8b-latest"' >> ~/.bashrc
source ~/.bashrc
cat > llm_demo.jac <<'JAC'
py import os;
py import byllm;

glob llm = byllm.Model(model_name=os.getenv("JAC_MODEL", "gemini/gemini-1.5-flash-8b-latest"));

def generate_password() -> str by llm();

sem generate_password = """
Generate a strong password (>=12 chars) with upper, lower, digit, and symbol.
""";

with entry {
    print(generate_password());
}
JAC

jac check llm_demo.jac
jac run llm_demo.jac
Model must include provider prefix: gemini/…

11) Tiny API server (optional)

cat > app.py <<'PY'
from fastapi import FastAPI
app = FastAPI()
@app.get("/")
def root(): return {"ok": True, "msg": "WSL + Jac ready"}
PY

uvicorn app:app --host 0.0.0.0 --port 8000
# open http://localhost:8000

12) Git + GitHub (SSH)

Configure Git

git config --global user.name  "Your Name"
git config --global user.email "you@example.com"

Create SSH key

ssh-keygen -t ed25519 -C "you@example.com"
cat ~/.ssh/id_ed25519.pub
# add to GitHub → Settings → SSH keys

First commit & push

git init
git add .
git commit -m "chore: WSL + Jac setup"
git branch -M main
git remote add origin git@github.com:<you>/<repo>.git
git push -u origin main
Multiple accounts? Add an SSH host alias in ~/.ssh/config and use it in your remote URL.

13) Daily workflow

cd ~/projects/jac-starter
source .venv/bin/activate
code .

# run Jac
jac run main.jac

# run LLM demo
export GEMINI_API_KEY="..."
export JAC_MODEL="gemini/gemini-1.5-flash-8b-latest"
jac run llm_demo.jac

14) Troubleshooting

jac: command not found → activate venv: source .venv/bin/activate and/or reinstall pip install jaclang.
byLLM provider error → ensure JAC_MODEL="gemini/…" and GEMINI_API_KEY is set.
VS Code can’t see venv → “Python: Select Interpreter” → pick .venv/bin/python.
Push rejected → if remote has a README, run git pull --rebase origin main then push.

Windows-only (no WSL)

winget install Python.Python.3.12 Git.Git Microsoft.VisualStudioCode -e
mkdir C:\dev\jac & cd C:\dev\jac
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -U pip jaclang byllm
notepad main.jac
# paste hello-jac, then:
jac run .\main.jac
If activation is blocked, run once in PowerShell: Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned
